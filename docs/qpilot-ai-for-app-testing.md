# Qpilot.AI Overview

Qpilot.ai is a revolutionary AI-powered test automation assistant that empowers testers to automate complex web and mobile applications effortlessly. With just natural language descriptions, Qpilot.ai generates flawless, real-time automation scripts in various programming languages like **Java**, **Python**, and **JavaScript**.

It runs tests both locally and on cloud-based labs, offering **instant feedback**, reducing manual errors, and accelerating testing cycles. Whether you're a seasoned QA engineer or a newcomer to automation, Qpilot.ai adapts to your workflow seamlessly. 

✅ It also includes **self-healing** features, meaning your scripts are auto-adjusted for updates or UI changes.

## Qpilot Modes

There are two main ways to use Qpilot.ai:

- **Generation Mode**: Create and execute test cases using simple English language prompts.
- **Run Mode**: Re-run saved test cases with updated scripts on devices or browsers.

## 1. Using the Generation Mode

The **Script Generation Mode** allows users to write test cases using plain English prompts, execute them live, and save the script for future reuse.

### ✅ Steps

1. **Login** with your registered email and password.
2. **Connect** to an available device on the platform.
3. Navigate to **All Actions → Qpilot Mode**.
4. Select the **Application** you want to test.
5. Enter the **App Package** and **App Activity**.
6. Provide the **Feature Name** and **Test Description** in English.
7. Optionally, click **Add Test Data** for usernames, passwords, etc.
8. Optionally, add **API Steps** to integrate API validation.
9. Click **Save**.
10. Choose or create a **Test Suite**.
11. Provide a **Test Case Name** and click **Save**.
12. Click the **Generate** button to execute.
13. Watch the **Live Execution** on the connected device or browser.
14. The test is automatically saved and accessible later from the Qpilot Dashboard.

---

## 2. Using the Run Mode

Run Mode lets users execute previously saved test cases on different devices or browsers.

### ✅ Steps

1. Go to the **Qpilot Dashboard**.
2. Find your saved test case under **All Tests**.
3. Click on the **+ (expand icon)** to view details.
4. Click the **Script** button to view the test logic.
5. Explore the script details:
    - Use the **Language Dropdown** to change between Python or JavaScript.
    - Use the **Copy icon** to copy the script.
    - Click **Live Session** to view previous executions.
    - Click **Run Test Case** to run the script on a new device/browser.
    - Use **Console** to view step-by-step logs.
    - View **Test Data** for variables passed during generation.

---

## 3. Limitations

- **Supported Browsers**: Chrome (Mac and Windows)
- **Supported Script Languages**: Python and JavaScript
